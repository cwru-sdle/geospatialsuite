# Detect HUC level from column name or data

Detect HUC level from column name or data

## Usage

``` r
detect_huc_level(column_name, data_values)
```

## Arguments

- column_name:

  Name of the HUC column

- data_values:

  Sample HUC values

## Value

Character string of HUC type (e.g., "huc08", "huc12")

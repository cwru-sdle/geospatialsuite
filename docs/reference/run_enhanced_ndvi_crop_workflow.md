# Run enhanced NDVI crop analysis workflow -

Enhanced NDVI workflow with quality filtering, temporal analysis, and
visualization. to handle test scenarios and provide robust error
handling.

## Usage

``` r
run_enhanced_ndvi_crop_workflow(config, output_folder = tempdir())
```

## Arguments

- config:

  Analysis configuration

- output_folder:

  Output directory

## Value

List with enhanced NDVI results and visualizations
